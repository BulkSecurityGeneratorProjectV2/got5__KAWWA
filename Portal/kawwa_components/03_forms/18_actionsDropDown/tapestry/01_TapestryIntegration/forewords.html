The ActionsDropDown component is an easy way to create a drop down list of actions. You just need to give a Map<String, List<String>> Object to this component. The component will use the keys of this Map for the category of actions (associated to the associated list).

By default, an eventLink will be displayed, triggering a KawwaEventsConstants.ACTIONSDROPDOWN_SELECT event. You can catch this event, and the corresponding method will have two parameters : the category and the selected item (formatted in a camel-case way).

You can override the rendering of each item, by defining a block parameter. The name of block should follow this rule : (category formatted in a camel-case way)_(item formatted in a camel-case way)
