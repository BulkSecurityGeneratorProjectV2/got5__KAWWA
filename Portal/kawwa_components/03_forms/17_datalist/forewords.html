This is a simple input pattern that doesn't require a CSS class.

The datalist HTML5 element, refered by the list attribute, is a handy and straightforward way to list predefined options suggested to the user.

The datalist may be used whenever the expected input entry value can be one of several predicted values. It doesn't mean, however, that the user is stuck with one of the values listed, but that she has a simple helper to fill out the field.
