Kawwa Example Templates

Default Theme / Version 1

Forewords

In this pack you will find some templates that may be used as starting points for your own pages. Feel free to modify them as you want!

With this pack you have:

  • 8 HTML files, in the "templates" folder, each one of them gathers together the components of a specific cathegory.
  • Example images and icons, in the "img" folder: feel free to use and modify them
  • 3 style-sheets:
    • "k-structure.css": resets the styles and defines the main page structure
    • "k-theme0.css": main styles for the theme
    • "i-hacks0.css": fixes for Internet Explorer, versions prior to the version 9

Your feedback is welcome! Send us your comments and suggestions to our mailing list dl-kawwa@atos.net.

Licenses

Web Fonts

Kawwa's default theme uses two web fonts that cannot be bundled ready to use with the templates, as defined by their End-User License Agreement, EULA.

If you want to use those fonts, you can access the site Font Squirrel to generate the font files needed:

  • Novecento Wide: this font is used for titles and navigation
  • PT Sans: this font is used for all body texts

To generate your web font files, click on the "@font-face Kit" link, on the font's main page.

Once generated the web font files, all you have to do is to add those files to your "css" folder. Font stacks are already defined for those fonts in your style-sheet.

Before use them, please check the license for each web font.
