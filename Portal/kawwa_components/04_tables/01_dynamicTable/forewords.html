This component uses the "DataTables" plug-in which allows you to add advanced interactive controls to your tables.

"DataTables" key features are:

  • Variable length pagination
  • On-the-fly filtering
  • multi-column sorting
  • Multiple data sources display

For more informations about how to translate the DataTable plug-in, please visit the official documentation.

Please take note that, altough very powerful and solid, this plug-in is not fully accessible (no ARIA scripting, no alternative aids, etc.) and it should not be used in a web project without taking those limits into account.

The main reason for adding it to our Kawwa2 components is that we've been informed that the plug-in are evolving and it will be make fully accessible. In the meantime, please consider carefully before using it.
