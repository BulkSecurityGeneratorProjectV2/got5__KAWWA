The demo below shows an example of how a k-product will be displayed in a catalog.

To have a product catalog, simply list your k-product components inside the catalog block.

In order to style the catalog products correctly, be sure to have on your style-sheet the code for the product template also.

Note that this component uses the HTML5 element article. If your site uses the XHTML DOCTYPE, just replace the HTML5 element by a div.
