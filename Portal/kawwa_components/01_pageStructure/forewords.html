Not exactly a component, the page structure will work as a matrix, a basic grid to all your Kawwa pages.

This page presents the following predefined blocks:

  • the page header: the page header will contain all generic content for a site. Consider always having the site's title there
  • the navigation bar: the navigation bar will hold the main navigation of the site (for example, the "mainNavigation" component)
  • the columns wrapper: this block "holds together" the different columns of the page.
    • the menu or secondary column (optional): here you have two options: you may add the #menu column if you want the first column of your page to display a second level of navigation, or you may add the #secondary column if the first column of your page will hold parallel content to the main one.
    • the main column (optional): this column will hold the main content of the page and, in the case of transversal content, it will hold the columns for main and transversal content
    • the sidebar (optional): this column is reserved for all content transversal to the main content of the page
