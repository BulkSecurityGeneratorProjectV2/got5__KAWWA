Use this component for you main site navigation. It can contain two hierarchical levels.

This same component can work in three different ways:

  • A simple one level navigation: in this case, you will have only one level of ul tags in your HTML source
  • A simple two levels navigation: if you have two levels of ul tags, you may add the "subnavbar" class to the first level ul tag ("ul.k-navbar"), and the second level of navigation will be displayed
  • Two levels navigation with dropdown: add the "dropdown" class to you second level ul tags
  • Two levels navigation with enhanced dropdown: in this case, you may have a div tag instead of a second level ul tag. This div tag can have multiple lists.
