productGallery (directive in module kawwa2 )

Description

Create an image gallery using the jqzoom jQuery plugin.

Usage

as attribute
<ANY product-gallery
     title="{string}"
     gallery="{Array of Object}"
     targets="{Object}">
   ...
</ANY>

Parameters

  • title – {string} –

    Give a title to your gallery

  • gallery – {Array of Object} –

    targets a variable from the scope of the controller. It is an array of object containing for each one : - title: the title of the image - thumb: the path to the thumblr image - small: the path to the small image - hd: the path to the big image

  • targets – {Object} –

    a variable from the scope of the controller. It allows you to overload the jqzoom jQuery plugin.
