fieldComment (directive in module kawwa2 )

Description

Show a tool tip when an action occur on the element. Use the Tipsy jQuery plugin

Usage

as attribute
<ANY field-comment="{Object}"
     title="{String}">
   ...
</ANY>

Parameters

  • fieldComment(optional) – {Object=} –

    allows you to overload the Tipsy jQuery plugin

  • title – {String} –

    The content of the tooltip
