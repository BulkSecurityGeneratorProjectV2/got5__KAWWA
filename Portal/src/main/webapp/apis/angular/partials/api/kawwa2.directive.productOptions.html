productOptions (directive in module kawwa2 )

Description

Enhances standard form elements like buttons, inputs and anchors to themeable buttons with appropriate hover and active styles. Use the widget button from jQuery-UI

Usage

as attribute
<ANY product-options="{Object}"
     name="{String}"
     products="{Array of Object}">
   ...
</ANY>

Parameters

  • productOptions(optional) – {Object=} –

    allows you to overload the button widget

  • name – {String} –

    is the attribut name use for the submission.

  • products – {Array of Object} –

    contains the infos for each buttons you want. These infos are - name {string} will be the value of the element selected - img {string} the path to the image -rel {string} for the img tag
