raty (directive in module kawwa2 )

Description

Create a Rating widget from radio input. It uses the rating jquery plugin.

Usage

as attribute
<ANY raty
     ngModel="{number}"
     [ratingOptions="{Object}"]>
   ...
</ANY>

Parameters

  • ngModel – {number} –

    Assignable angular rating to data-bind to.

  • ratingOptions(optional) – {Object=} –

    allows you to overload the rating jquery plugin
