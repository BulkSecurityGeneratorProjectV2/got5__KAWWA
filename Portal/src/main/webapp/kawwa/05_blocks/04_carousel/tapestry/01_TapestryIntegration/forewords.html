NB: Those components belong to the tapestry5-jquery project, please have a look to the official website.

Carousel parameters
Name Required Java Type Default Prefix Default Value Description
clientId false String literal The client id of the carousel
className false String literal The class attribute is added to the carousel
params false JSONObject prop The slider parameters (please refer to jcarousel documentation)

You have to use an other component inside a carousel component, one carouselitem per image you want to show in the carousel. It depends on the carousel component and cannot be used alone.

CarouselItem parameters
Name Required Java Type Default Prefix Default Value Description
width false int literal 75 The width of the image in pixels
height false int literal 75 The height of the image in pixels
imageSource true String prop The path of the image
page false String literal If you provide a value, the image will be surrounded by a pagelink which targets this page.
event false String literal If you provide a value, the image will be surrounded by an eventlink which triggers this event.
context false Object prop If the image is surrounded by a link, you can provide a context thanks to this parameter.
zone false String literal If you provide a zone name, the surrounding link will be an ajax link
